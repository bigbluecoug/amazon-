Campaign Sender Studio

Prepare your gifting workflow for Amazon Business automation.

Plan the sequence, assign recipients, configure Amazon Business placeholders, and process due gifts through a review-safe order pipeline.

1. Campaign

Name the campaign, choose a start date, and define the audience rules.

Campaign changes save locally in this project.

2. Send sequence

Add as many steps as you want. Each step can have a delay, product, quantity, and personalized message.

3. Recipients

Paste recipients manually, then assign each contact to the right owner.

HubSpot can be added later. For now, paste recipients and assign them here.

4. Amazon Setup

Save the Amazon Business integration details now so the backend shape is ready when credentials are available.

Credentials are stored locally for now. Live order submission is still a stub until you have approved Amazon Business API access.

5. Email Automation

Configure the email provider details and choose whether the email should send when the gift is marked sent or marked delivered.

Emails will only send automatically after SMTP settings are filled in and an order changes to the trigger status.

6. Execution

Build the queue, process gifts that are due, and maintain local order history before live submission is switched on.

Direct Amazon checkout is not enabled by default. Due gifts are processed into tracked local order records first.

7. Order history

Track due gifts, submission readiness, and manual status changes.

8. Queue preview

Review the recipients, schedule, messages, and Amazon-ready payloads.
